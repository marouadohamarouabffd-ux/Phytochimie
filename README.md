# Phytochimie
Les alcaloïdes sont des molécules à bases azotées, le plus souvent hétérocycliques, très majoritairement d'origine végétale. Ils peuvent se présenter sous forme de molécules organiques hétérocycliques azotées basiques.


<img width="220" height="181" alt="image" src="https://github.com/user-attachments/assets/a63e91fa-ba7c-4b46-8d70-c3560dd0d150" />


Naturels	Morphine, codéine, thébaïne, papavérine, noscapine
Semi-synthétiques	Hydromorphone, hydrocodone, héroïne
Synthétiques	Fentanyl, péthidine, méthadone, propoxyphène
